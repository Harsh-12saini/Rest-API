# Rest-API
A REST API project built with Node.js and Express, implementing CRUD operations using GET, POST, PUT, PATCH, and DELETE methods.
## Features
- GET API
- POST API
- PUT API
- PATCH API
- DELETE API
- Express Routing
- Middleware
- Error Handling
## Tech Stack
- Node.js
- Express.js
- JavaScript
- REST API
- Git
- GitHub
- Postman
## Installation

```bash
npm install
```

```bash
node app.js
